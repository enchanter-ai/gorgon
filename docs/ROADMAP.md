# Gorgon Roadmap

- [x] Five engines G1-G5 (stdlib only).
- [x] Six sub-plugins + one `full` meta-plugin.
- [x] Four hook bindings (SessionStart, UserPromptSubmit, PostToolUse, PreCompact).
- [x] Three managed agents (Haiku / Sonnet / Opus).
- [x] Honest-numbers contract on every advisory.
- [x] G5 Gauss-accumulation drift posterior.
- [x] Phase-1 file-tail event-bus publish via Pech `publish.py`.
